# Лабораторна робота №6: Обробка відеопотоку за допомогою OpenCV

##  Опис завдання
Розробка C++ додатку для зчитування відеопотоку в середовищі WSL2 (Ubuntu), обробка кадрів у реальному часі за допомогою фільтра Кенні (Canny) для виділення контурів, а також реалізація графічного інтерфейсу (GUI) з Trackbar для динамічної зміни чутливості фільтра.

## 💻 Вимоги до ОС та компілятора
* **Операційна система:** Ubuntu (під керуванням WSL2)
* **Графічна підсистема:** Налаштований X-сервер на хості (наприклад, VcXsrv) для відображення вікон OpenCV (експорт `DISPLAY=:0`)
* **Компілятор:** GCC/G++ версії 11 або вище (підтримка стандарту C++17)
* **Система збірки:** CMake версії 3.20 або вище
* **Залежності:** Бібліотека OpenCV 4.x (модулі `core`, `imgproc`, `highgui`, `videoio`) та утиліта `ffmpeg`

## 🚀 Інструкція з налаштування та запуску

Усі рутинні операції автоматизовані за допомогою командних скриптів Bash. Перед запуском переконайтеся, що вони мають права на виконання (`chmod +x *.sh`).

### Крок 1: Встановлення залежностей системи
Скрипт оновлює індекси пакетів та встановлює необхідні бібліотеки для роботи OpenCV та мультимедіа:
```bash
./preinstall.sh

Компіляція проєкту
Скрипт автоматично створює директорію збірки build, запускає генерацію файлів CMake та компілює бінарний файл додатку:


./build.sh

апуск лабораторної роботи
Скрипт експортує змінні середовища для графічного сервера та запускає готовий скомпільований файл:


./run.sh
